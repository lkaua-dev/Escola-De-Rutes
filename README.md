# 💜 Escolas de Ruths - Landing Page

Bem-vindo ao repositório do site **Escolas de Ruths**! 

Este projeto é uma landing page desenvolvida para um evento da nossa igreja. O objetivo principal do site é apresentar o propósito do encontro, exibir momentos especiais através de uma galeria de fotos e facilitar a inscrição dos participantes diretamente pelo WhatsApp.

## 🚀 Funcionalidades

- **Cabeçalho Duplo:** Espaço dedicado para as logos da Igreja e do evento.
- **Sobre o Evento:** Seção explicativa sobre o que é e como funciona o "Escolas de Ruths" (texto atualizável).
- **Inscrição Facilitada:** Botão de *Call to Action* (CTA) que redireciona o usuário para o WhatsApp, já com uma mensagem padrão de inscrição preenchida.
- **Galeria de Imagens:** Carrossel/Grid de fotos integrados na página para exibir momentos do evento.
- **Álbum Externo:** Botão "Ver mais fotos" com redirecionamento para o álbum completo na galeria da Samsung.
- **Redes Sociais:** Link direto para acompanhamento do perfil no Instagram.
- **Rodapé (Footer):** Informações de direitos autorais e créditos de desenvolvimento.

## 🎨 Design e UI/UX

A identidade visual foi pensada para ser moderna, acolhedora e responsiva:
- **Paleta de Cores:** Roxo Claro, Roxo Escuro e Branco.
- **Animações:** Transições suaves e animações fluidas ao rolar a página (scroll) e ao passar o mouse sobre os botões (hover).
- **Responsividade:** Layout 100% otimizado para dispositivos móveis (smartphones e tablets) e desktops.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias base da web, sem necessidade de frameworks pesados, garantindo alta velocidade de carregamento:

- **HTML5** (Estruturação semântica)
- **CSS3** (Estilização, Flexbox/Grid, responsividade e animações)
- **JavaScript (Vanilla)** (Interatividade da galeria e animações dinâmicas)
